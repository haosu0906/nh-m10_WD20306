# My Static Site

This project is a simple static website consisting of multiple pages and reusable components. Below is an overview of the project structure and its contents.

## Project Structure

```
my-static-site
├── index.html        # Main entry point of the website
├── about.html        # About page
├── contact.html      # Contact page
├── components        # Reusable components
│   ├── header.html   # Header component
│   └── footer.html   # Footer component
├── css               # Stylesheets
│   ├── main.css      # Main styles for the website
│   └── variables.css  # CSS variables for consistent styling
├── .gitignore        # Files and directories to ignore in version control
└── README.md         # Documentation for the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the homepage.
3. Navigate to the other pages using the links provided in the header.

## Overview

This static site is designed to provide information about a fictional company, including details on its services, an about page, and a contact form. The site is styled using CSS, with variables defined for easy customization. The header and footer components are included in each page to maintain a consistent layout.
